# 📊 👨‍👩‍👦 dpat-group-project
2025 2학년 1학기 전공선택 데이터처리와분석기술 기말과제 팀플 자료

## ❓ About Project
이 프로젝트는 뉴스와 비트코인의 상관관계를 분석하고 자동 매매 봇을 만드는 것을 목표로 합니다. 데이터는 2024년 데이터를 사용합니다.

[`src`](/src)폴더에 대한 설명은 다음과 같습니다.   
[`src/coin`](/src/coin/): 비트코인 데이터 수집, 필터링, 분석, 시각화
[`src/news](/src/news/): 뉴스 데이터 수집


## 📝 데이터 분석 계획
1. 2024년 비트코인 데이터를 수집한다. (고가, 저가, 날싸 등등)
2. 등략률(절댓값)이 5%이상 이 되는 부분만 필터링한다.
3. 필터링 한 데이터를 토대로 뉴스데이터를 수집한다
4. 수집한 뉴스데이터에서 중복되는 키워드를 추출한다
5. 키워드의 빈도수를 워드 클라우드나 막대그래프로 시각화한다.
